## Getting Started

### To Access my project locally you can follow these steps

1. Enter this url:  https://github.com/jorgegoco/To-Do-List.git in your web browser.
2. Once opened navigate to the top left level of the project a green code download button will be visible on the righthand side.
3. Select download Zip option from drop down menu.
4. Once the download is complete you will be able to access my project locally.
Here is my repository link:  https://github.com/jorgegoco/To-Do-List.git
